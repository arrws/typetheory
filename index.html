    switches

    Designed for high durability mechanical Kailh BOX switches, IP56-certified against dust and water. We can install any Kailh BOX switches, but shipping times will vary based on availability of the switches.

    We try to keep our favourite switches in stock, these are:

    • Kailh Pale Blue (medium volume click, 60 cN actuation), similar to a Cherry MX Blue.
    • Kailh Burnt Orange (silent tactile, 70 cN actuation), similar to heavier Cherry MX Brown or a Cherry MX clear.

    Kaycaps

    Blank low-profile DSA keycaps offered in a variety of customizable colours. These keycaps have a uniform profile, which offers the best typing experience when combined with the sculpted concave shape of the keyboard. Prefer a different keycap profile? You can order the keyboard without keycaps and use any MX-compatible or Kailh BOX-compatible set of keycaps.

    Electronics

  • The switches are connected using a bespoke flexible PCB, greatly improving the keyboard's reliability compared to hand-wired Dactyl-Manuforms. (ai putea adauga flexible PCB and amoeba for thumb clusters to be fair).
  • Powered by two Puchi-C controllers featuring the popular ATmega32U4 microcontroller and a strong USB-C port.
  • Each half has a USB-C port and a 3.5mm jack. The two halves are connected using a headphone cable. Only one of the halves needs to be connected to the computer.

    Software

    TypeTheory runs the highly customizable open source QMK firmware.

    The default layout is inspired by the QMK default Dactyl-Manuform 5x6 layout, but this can easily be changed. You can send a message with details of your preferred layout, or you can download the firmware source code from our Github and customize it to your needs. Each side of the keyboard is able to function alone, this allows using only the left side of the keyboard plus a mouse when gaming.

    Fair Prices Instructions for ordering

    TypeTheory is highly customizable, you should send a message with your order that includes your preferences for the following:

    1. Case colour can be white, grey, or black.
    2. Switches can be Kailh BOX Burnt Orange, Kailh BOX Pale Blue, or any other Kailh BOX switches.
    3. Keycap colours are explained in one of the images, you should mention (for each hand):
      • Thumb, CMD, SP zones can each be black, grey, or white.
      • Default zone can be black, grey, or white.
      • Num, Home, Extra, and Mod zones can each be black, grey, white, green, blue, purple, pink, or red.

    If you don't specify one of these preferences, we will make a grey case, and install burnt orange switches as well as all black keycaps.
